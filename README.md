# Side Quest Week 10 - Bun Run: Escape the Wolf

## Setup and Interaction Instructions

To run the sketch, open `index.html` in Google Chrome using Live Server.

The goal is to trying to get to the other side to the door to go to the next level without getting caught by the wolf and avoid the traps. You will be given 3 chances if you use up all 3 lives you restart the whole game. There will be apples displayed in the gaem but it si not necessary to collect all of them to win the whole game. 

**Controls**
- A / Left Arrow: move left
- D / Right Arrow: move right
- W / Up Arrow / Space: jump
- R: reset the current attempt
- Enter: start or continue

**Platform types**
- `ground`
- `normal`
- `bounce`
- `moving`
- `falling`

**Trap types**
- `hiddenSpikes`
- `fakeStrawberry`
- `rollingLog`
- `fallingBranch`

There is a debug panel. To use it press **E** that will show at the bottom of the screen. Then you can press the key that correspond to the shortcuts. 

**Keyboard shortcuts:**
- 1, 2: jump to that level 
- S or s: jump to the start screen 
- W or w: jump to the win screen (All Levels)
- O or o: jumpp to the restart screen
